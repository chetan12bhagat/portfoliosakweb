# Sakshi Thakre Portfolio

A premium portfolio website showcasing projects in Web Development and IoT.

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- React Three Fiber (3D Elements)

## Features
- Interactive 3D Robot Assistant
- Responsive Design
- Project Showcases
- Contact Section

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/chetan12bhagat/portfoliosakweb.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```
